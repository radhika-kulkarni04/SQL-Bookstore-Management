SQL Bookstore Management System
Project Overview

This project demonstrates how to design and manage a Bookstore Database using PostgreSQL and pgAdmin 4. It covers database creation, importing datasets, and writing SQL queries for analysis.

Database Structure

Books – Contains book details such as title, author, and price.

Customers – Stores customer information like name, email, and city.

Orders – Records book purchases made by customers.
